使用 clang-format 對程式碼進行排版

Posted on 2019-05-27 | Comments:

介紹

clang-format 是基於 clang 的程式碼格式化（format，與硬碟的格式化不同，指針對程式碼風格進行排版）工具，支持許多種 Coding Style 如：

  • Google Style Guides
  • LLVM Coding Standards
  • Webkit Code Style Guidelines
  • Mozilla Coding style
  • …

安裝

Microsoft Windows 10

在 Windows 作業系統下，必須先安裝好 Python 環境並將其添加至環境變數中，接著在 LLVM Snapshot Builds 頁面選擇 Windows 版本進行安裝，注意在安裝過程中要勾選 Add LLVM to the system PATH for all users 或 Add LLVM to the system PATH for current user 其中一項。安裝完後可以在終端機中檢查當前版本：
