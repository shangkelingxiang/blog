刪除過往 Github Repo 的提交紀錄

Posted on 2018-03-12 | Comments:

前言

Github 是著名的代碼託管網站，配合版本控制軟體 Git 可以很好地記錄提交紀錄，但有時因為沒有經驗或是純粹開始作為玩玩而已，會有許多提交紀錄有點「醜」…

解決方法

透過以下 Git Command 可以將過往的提交紀錄刪除，並保持當前的代碼狀態：

git checkout --orphan
